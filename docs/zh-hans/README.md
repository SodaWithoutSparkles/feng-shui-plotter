# 风水平面标注器 — 简介 ✅

为风水师设计的网页工具，可在平面图上快速标注与检视飞星（Fly Star），不需要专门软件（只需浏览器）。

- 在线试用: https://sodawithoutsparkles.github.io/feng-shui-plotter/ 🔗

---

## 主要功能 🌟

- **内置飞星计算**：自动生成飞星图并支持年份切换与颜色参考。
- **专业罗盘**：多种显示与编辑模式，支持“锁定朝南（Lock to South）”以便方位校正。📐
- **简化的标注流程**：以对象为单位操作，无需复杂图层管理。
- **导入平面图**：支持拖拽或粘贴图片（PNG/JPG）。
- **导出高质量图片**：支持导出画布（PNG 等）。
- **自动保存与项目存档**：使用压缩方式保存到 localStorage，方便载入与分享。
- **免费、跨平台、即开即用**：适合在现场或办公室使用。

---

## 快速上手 🚀

1. 打开在线版本：
   - https://sodawithoutsparkles.github.io/feng-shui-plotter/
2. 导入平面图（拖放或文件导入），放置并旋转罗盘以匹配建筑方位。
3. 打开右侧「飞星」面板查看飞星分布；使用绘图工具添加注记。
4. 完成后通过「File → Export」导出图片用于报告或打印。
